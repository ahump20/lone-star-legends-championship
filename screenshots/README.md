## Guidelines

### Screenshot Specs
- **Format**: PNG (for quality)
- **Mobile**: 720x1280 (9:16 aspect ratio)
- **Desktop**: 1280x720 (16:9 aspect ratio)
- **File size**: <500KB each (compressed)

### Content Guidelines
- Show actual gameplay (not mockups)
- Include UI elements
- Display key features
- Use high-quality graphics settings
- Avoid personal information
- Show variety (different stadiums, characters, modes)

## Taking Screenshots

### Mobile Device
1. Open game on mobile device
2. Navigate to key screens
3. Use device screenshot function:
   - iOS: Volume Up + Power button
   - Android: Volume Down + Power button
4. Transfer to computer

### Desktop Browser (Mobile View)
1. Open Chrome/Firefox DevTools
2. Enable device toolbar (Responsive Design Mode)
3. Set dimensions to 720x1280
4. Capture with:
   - Chrome: DevTools > Capture screenshot
   - Firefox: DevTools > Take a screenshot
   - Or use system screenshot tool

### Recommended Scenes

## Optimization

After capturing:
1. Crop to exact dimensions (720x1280)
2. Compress with tools:
   - TinyPNG (online)
   - ImageOptim (Mac)
   - PNGGauntlet (Windows)
3. Verify quality still looks good
4. Keep under 500KB per file
